📚 CA Finals 2027 Preparation Tracker
> A comprehensive preparation dashboard built exclusively for **CA Final
> (ICAI New Scheme 2023)** students.
This tracker combines syllabus management, lecture tracking, revision
planning, study session logging, analytics, goal tracking, and cloud
synchronization into a single application. It is designed to replace
multiple spreadsheets, notebooks and standalone apps with one
centralized dashboard.

Cloud Sync
Cloud Sync uses GitHub Gist.
Why use it?
Without Cloud Sync:
Progress remains only on the current browser.
With Cloud Sync:
Laptop
Desktop
Mobile
all remain synchronized.
---
Initial Setup
Generate a GitHub Personal Access Token with Gist permission.
Open Cloud Sync (☁️).
Paste the token.
Perform the first Sync Up.
The tracker automatically creates a private GitHub Gist.
---
Sync Up
Uploads:
Progress
Study sessions
Revision status
Settings
to GitHub.
---
Sync Down
Downloads the latest data from your GitHub Gist and restores it locally.

FAQ
Will I lose my progress?
No.
Progress is automatically saved locally.
If Cloud Sync is configured, it can also be restored from GitHub.
---
Can I use multiple devices?
Yes.
Use:
Sync Up
Sync Down
to synchronize your progress.
---
Does the tracker require an internet connection?
No.
Only Cloud Sync requires internet.
Everything else works offline.
---
Troubleshooting
Sync Down does not show recent changes
Ensure you performed Sync Up on the other device.
Verify both devices use the same GitHub account and Gist.
Website not updating
Refresh using:
Ctrl + Shift + R
or clear browser cache.
Progress missing
Use Sync Down if Cloud Sync has been configured.
